Cauldron harnesses a range of text tokens to ensure a polished and engaging user experience.

## Font

Font tokens tie an intention to a font family.

<CssParamsTable
  param={'--base-font-family'}
  formatName={() => 'Base'}
  renderExample={(name, value) => (
    <span style={{ fontFamily: value }}>
      This is an example of the font family
    </span>
  )}
/>

## Font size

Font size tokens are relative to the **--text-size-small**. Use font size tokens to define the size of the font.

<CssParamsTable
  param={'--text-size'}
  renderExample={(name, value) => <span style={{ fontSize: value }}>Text</span>}
/>

## Font weight

Font weight tokens define the weight of the font face.

<CssParamsTable
  param={'--font-weight'}
