import { RadioCardGroup, FieldWrap } from '@deque/cauldron-react'

```jsx
import { RadioCardGroup } from '@deque/cauldron-react'
```

<Note>
  Input components _should_ be wrapped in a [FieldWrap](./FieldWrap). The
  `RadioCardGroup` component has been designed specifically to meet color
  contrast requirements against a `FieldWrap` background.

  If you use a `RadioCardGroup` without a `FieldWrap` (for example, within a
  `Modal`, where `FieldWrap`'s `Panel` appearance might not be appropriate),
  you _must_ verify that it meets accessibility requirements in all component
  states, especially for color contrast.
</Note>


## Examples

```jsx example
<FieldWrap>
  <div className="Field__label" id="radio-card-group-label">Do you like coffee?</div>
  <RadioCardGroup
    defaultValue="yes"
    aria-labelledby="radio-card-group-label"
    radios={[
      {
        id: 'yes',
        value: 'yes',
        label: 'Yes',
        cardImg: (
          <div
            style={{
              backgroundColor: 'green',
              height: 60,
              width: 60,
              borderRadius: 60
            }}
          ></div>
        ),
        cardIcon: 'check-circle'
      },
      {
        id: 'no',
        value: 'no',
        label: 'No',
        disabled: true,
        cardImg: (
          <div
            style={{
              backgroundColor: 'red',
              height: 60,
              width: 60,
              borderRadius: 60
            }}
          ></div>
        ),
        cardIcon: 'check-circle'
      },
      {
        id: 'still-yes',
        value: 'still yes',
        label: 'Still Yes',
        cardImg: (
          <div
            style={{
              backgroundColor: 'green',
              height: 60,
              width: 60,
              borderRadius: 60
            }}
          ></div>
        ),
        cardIcon: 'check-circle'
      }
    ]}
  />
</FieldWrap>
```
