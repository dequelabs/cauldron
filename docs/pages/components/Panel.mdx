  PanelContent,
  Toast,

  Code

```js
import { Panel, PanelHeader, PanelContent } from '@deque/cauldron-react';
```

## Examples

### Panel

```jsx example
<Panel>
  <p>Panel Content</p>
</Panel>
```

### Panel with Heading (Prop)

```jsx example
<Panel heading={{ id: 'example', text: 'Panel Heading', level: 3 }}>
  <p>Panel Content</p>
</Panel>
```

### Panel (Composed)

<Note>
  The `PanelHeader` should not be used with the `Panel` heading prop at the same
  time. Choose one or the other to use.
</Note>

```jsx example
<Panel>
  <PanelHeader>
    <h3>Panel Header</h3>
  </PanelHeader>
  <PanelContent>
    <p>Panel Content</p>
  </PanelContent>
</Panel>
```

### Disabling Panel Padding

```jsx example
<Panel
  heading={{ id: 'example', text: 'Panel Heading', level: 3 }}
  padding={false}
>
  <p style={{ paddingLeft: '1rem' }}>Panel Content</p>
</Panel>
```

### Disabling Panel Content Padding

```jsx example
<Panel>
  <PanelHeader
    style={{
      display: 'flex',
      justifyContent: 'space-between',
      alignItems: 'center',
      flexWrap: 'wrap'
    }}
  >
    <h3>Panel Header</h3>
    <Button variant="primary" thin>
      <Icon type="plus" />
      Button
    </Button>
  </PanelHeader>
  <PanelContent
    padding={false}
    style={{
      columns: '2',
      columnRule: 'var(--panel-divider-border)',
      columnGap: '0'
    }}
  >
    <div style={{ padding: 'var(--space-small)' }}>Column 1</div>
    <div style={{ padding: 'var(--space-small)' }}>Column 2</div>
  </PanelContent>
  <PanelContent>
    <p>Footer</p>
  </PanelContent>
</Panel>
```

### Header With Flexbox Content

```jsx example
<Panel padding={false}>
  <PanelHeader
    style={{
      display: 'flex',
      justifyContent: 'space-between',
      alignItems: 'center',
      flexWrap: 'wrap'
    }}
  >
    <h3>Panel Header</h3>
    <Button variant="primary" thin>
      <Icon type="plus" />
      Button
    </Button>
  </PanelHeader>
  <PanelContent padding>
    <p>Panel Content</p>
  </PanelContent>
</Panel>
```

### Stacked Content Sections

Stacking `PanelContent` components will create a visual separation between the sections.

```jsx example
<Panel>
  <PanelHeader
    style={{
      display: 'flex',
      justifyContent: 'space-between',
      alignItems: 'center',
      flexWrap: 'wrap'
    }}
  >
    <h3>Panel Header</h3>
    <Button variant="primary" thin>
      <Icon type="plus" />
      Button
    </Button>
  </PanelHeader>
  <PanelContent>
    <p>
      Lorem ipsum dolor sit amet, consectetur adipiscing elit. Nam
      ornareadfasdfdsaf eleifend tempus. Sed odio sem, pulvinar nec erat ac,
      iaculis maximus nisl. Pellentesque gravida lorem sit amet ipsum efficitur,
      eget finibus justo feugiat. Suspendisse blandit, urna eu facilisis
      pulvinar, mi turpis mattis magna, at rutrum leo leo egestas risus.
      Curabitur mattis in erat vel tristique. Nam fermentum consectetur ipsum,
      ut placerat purus volutpat vitae. Nam molestie vulputate accumsan. Sed
      vitae odio eu justo venenatis laoreet.
    </p>
  </PanelContent>
  <PanelContent style={{ textAlign: 'center' }}>
    <Button variant="secondary">Previous</Button>
    <Button>Next</Button>
  </PanelContent>
</Panel>
```
