## Examples

### Simple (Composable Heading and Content)

<Note>
  The `PanelHeader` should not be used with the `Panel` heading prop at the same
  time. Choose one or the other to use.
</Note>

There are two ways to include headings in the `Panel` component. The first is to compose the heading with `PanelHeader`
and `PanelContent` components. The second is to use the `heading` prop on the `Panel` component.

```jsx example
<Panel>
  <PanelHeader>
    <h3>Panel Header</h3>
  </PanelHeader>
  <PanelContent>
    <p>Panel Content</p>
  </PanelContent>
</Panel>
```

### Simple (Heading Prop)

```jsx example
<Panel heading={{ id: 'example', text: 'Panel Heading', level: 3 }}>
  <p>Panel Content</p>
</Panel>
```

### Header with Flex Heading Content

```jsx example
<Panel padding={false}>
  <PanelHeader
    style={{
      display: 'flex',
      justifyContent: 'space-between',
      alignItems: 'center',
      flexWrap: 'wrap'
    }}
  >
    <h3>Panel Header</h3>
    <Button variant="primary" thin>
      <Icon type="plus" />
      Button
    </Button>
  </PanelHeader>
  <PanelContent padding>
    <p>Panel Content</p>
  </PanelContent>
</Panel>
```

### Stacked Content Sections

Stacking `PanelContent` components will create a visual separation between the sections.

<Note>
  Padding must be disabled via `<PanelContent padding={false} />` when stacking
  the sections.
</Note>

```jsx example
<Panel padding={false}>
  <PanelHeader
    style={{
      display: 'flex',
      justifyContent: 'space-between',
      alignItems: 'center',
      flexWrap: 'wrap'
    }}
  >
    <h3>Panel Header</h3>
    <Button variant="primary" thin>
      <Icon type="plus" />
      Button
    </Button>
  </PanelHeader>
  <PanelContent>
    <p>
      Lorem ipsum dolor sit amet, consectetur adipiscing elit. Nam
      ornareadfasdfdsaf eleifend tempus. Sed odio sem, pulvinar nec erat ac,
      iaculis maximus nisl. Pellentesque gravida lorem sit amet ipsum efficitur,
      eget finibus justo feugiat. Suspendisse blandit, urna eu facilisis
      pulvinar, mi turpis mattis magna, at rutrum leo leo egestas risus.
      Curabitur mattis in erat vel tristique. Nam fermentum consectetur ipsum,
      ut placerat purus volutpat vitae. Nam molestie vulputate accumsan. Sed
      vitae odio eu justo venenatis laoreet.
    </p>
  </PanelContent>
  <PanelContent style={{ textAlign: 'center' }}>
    <Button variant="secondary">Previous</Button>
    <Button>Next</Button>
  </PanelContent>
</Panel>
```

### Overwriting Padding

```jsx example
<Panel padding={false}>
  <PanelHeader
    style={{
      display: 'flex',
      justifyContent: 'space-between',
      alignItems: 'center',
      flexWrap: 'wrap'
    }}
  >
    <h3>Panel Header</h3>
    <Button variant="primary" thin>
      <Icon type="plus" />
      Button
    </Button>
  </PanelHeader>
  <PanelContent
    padding={false}
    style={{ columns: '2', columnRule: '1px solid #ccc', columnGap: '0' }}
  >
    <div style={{ padding: '1rem' }}>Column 1</div>
    <div style={{ padding: '1rem' }}>Column 2</div>
  </PanelContent>
</Panel>
```
