import { RadioGroup, FieldWrap, Code } from '@deque/cauldron-react'

```js
import { RadioGroup } from '@deque/cauldron-react'
```

<Note>
  Use [FieldWrap](./FieldWrap) to wrap all input components. The `RadioGroup` component has been designed specifically to only be used when rendered as a child of this component, and may not be completely accessible if rendered outside of `FieldWrap`.
</Note>

## Examples

### Default

```jsx example
<FieldWrap>
  {/* This label is visually rendered, but you could also just use groupLabel below. */}
  <div className="Field__label" id="gyros-label">Do you like gyros?</div>
  <RadioGroup
    name="gyros"
    aria-labelledby="gyros-label"
    groupLabel="Gyros Question"
    radios={[
      {
        id: 'gyros-yes',
        label: 'Yes!!',
        value: 'yes'
      },
      {
        id: 'gyros-no',
        label: 'Nope.',
        value: 'no'
      },
      {
        id: 'gyros-friday',
        label: 'Only on Fridays',
        value: 'friday'
      }
    ]}
    defaultValue="yes"
  />
</FieldWrap>
```

### With Descriptions

```jsx example
<FieldWrap>
  <div className="Field__label" id="pizza-label">Do you like pizza?</div>
  <RadioGroup
    aria-labelledby="pizza-label"
    defaultValue="tuesday"
    name="pizza"
    radios={[
      {
        id: 'yes',
        label: 'Yes',
        labelDescription: 'Of course you like pizza',
        value: 'yes'
      },
      {
        disabled: true,
        id: 'no',
        label: 'No',
        labelDescription: 'Impossible, everyone likes pizza',
        value: 'no'
      },
      {
        id: 'tuesday',
        label: 'Only on Tuesdays',
        labelDescription: 'Pizza Tuesdays are rad',
        value: 'tuesday'
      }
    ]}
  />
</FieldWrap>
```

### Inline Layout

If your list of options is short, `RadioGroup` can optionally accept an `inline` prop to display the radio items in a horizontal orientation instead of vertically.

```jsx example
<FieldWrap>
  <div className="Field__label" id="robot-label">Are you a robot?</div>
  <RadioGroup
    aria-labelledby="robot-label"
    groupLabel="Robot Check"
    inline
    name="robot"
    radios={[
      {
        id: 'robot-yes',
        label: 'Yes',
        value: 'yes'
      },
      {
        id: 'robot-no',
        label: 'No',
        value: 'no'
      }
    ]}
  />
</FieldWrap>
```

### Descriptions and Disabled Radios

Each radio supports a labelDescription that displays an extra line of text beneath its label, useful for clarifications or hints. The disabled property disallows selection.

```jsx example
<FieldWrap>
  <RadioGroup
    groupLabel="Movie Genres"
    name="genres"
    defaultValue="comedy"
    radios={[
      {
        id: 'comedy',
        label: 'Comedy',
        value: 'comedy',
        labelDescription: 'Make me laugh.'
      },
      {
        id: 'horror',
        label: 'Horror',
        value: 'horror',
        labelDescription: 'Scare me!'
      },
      {
        id: 'romance',
        label: 'Romance',
        value: 'romance',
        labelDescription: 'Love is in the air.',
        disabled: true       // This option is not selectable
      }
    ]}
  />
</FieldWrap>
```
