import { RadioGroup, FieldWrap, FieldGroup, Code } from '@deque/cauldron-react'

```js
import { RadioGroup } from '@deque/cauldron-react'
```

<Note>
  Input components _should_ be wrapped in a [FieldWrap](./FieldWrap). The
  `RadioGroup` component has been designed specifically to meet color contrast
  requirements against a `FieldWrap` background.

  If you use a `RadioGroup` without a `FieldWrap` (for example, within a
  `Modal`, where `FieldWrap`'s `Panel` appearance might not be appropriate),
  you _must_ verify that it meets accessibility requirements in all component
  states, especially for color contrast.
</Note>

## Examples

### Default

```jsx example
<FieldWrap 
  as={FieldGroup} 
  label="Do you like gyros?"
>
  <RadioGroup
    name="gyros"
    radios={[
      {
        id: 'gyros-yes',
        label: 'Yes!!',
        value: 'yes'
      },
      {
        id: 'gyros-no',
        label: 'Nope.',
        value: 'no'
      },
      {
        id: 'gyros-friday',
        label: 'Only on fridays',
        value: 'friday'
      }
    ]}
    value="yes"
  />
</FieldWrap>
```

### With Descriptions

```jsx example
<FieldWrap 
  as={FieldGroup} 
  label="Do you like pizza?"
>
  <RadioGroup
    defaultValue="tuesday"
    name="pizza"
    radios={[
      {
        id: 'yes',
        label: 'Yes',
        labelDescription: 'Of course you like pizza',
        value: 'yes'
      },
      {
        disabled: true,
        id: 'no',
        label: 'No',
        labelDescription: 'Impossible, everyone likes pizza',
        value: 'no'
      },
      {
        id: 'tuesday',
        label: 'Only on Tuesdays',
        labelDescription: 'Pizza Tuesdays are rad',
        value: 'tuesday'
      }
    ]}
  />
</FieldWrap>
```

### Inline Layout

If your list of options is short, `RadioGroup` can optionally accept an `inline` prop to display the radio items in a horizontal orientation instead of vertically.

```jsx example
<FieldWrap 
  as={FieldGroup} 
  label="Are you a robot?"
>
  <RadioGroup
    defaultValue="no"
    name="robot"
    inline
    radios={[
      {
        id: 'robot-yes',
        label: 'Yes',
        value: 'yes'
      },
      {
        id: 'robot-no',
        label: 'No',
        value: 'no'
      }
    ]}
  />
</FieldWrap>
```
