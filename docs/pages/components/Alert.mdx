## Examples

### Default Alert

Display an alert dialog that prompts the user to take action, by default this alert is non-dismissable:

```jsx example
function DefaultAlertExample() {
  const [show, setShow] = useState(false)
  const toggleAlert = () => setShow(!show)
  return (
    <>
      <Button variant="secondary" onClick={toggleAlert}>Toggle Alert</Button>
      <Alert heading="Default Alert" show={show}>
        <AlertContent>
          Just letting you know that something happened.
        </AlertContent>
        <AlertActions>
          <Button onClick={toggleAlert}>Ok</Button>
          <Button variant="secondary" onClick={toggleAlert}>Close</Button>
        </AlertActions>
      </Alert>
    </>
  )
}
```

### Warning Alert

If the user action is potentially dangerous, use a warning variant to indicate potentially destructive actions:

```jsx example
function WarningAlertExample() {
  const [show, setShow] = useState(false)
  const toggleAlert = () => setShow(!show)
  return (
    <>
      <Button variant="secondary" onClick={toggleAlert}>Toggle Warning Alert</Button>
      <Alert variant="warning" heading="Danger Zone" show={show}>
        <AlertContent>
          Welcome to the danger zone!
        </AlertContent>
        <AlertActions>
          <Button variant="danger" onClick={toggleAlert}>Danger</Button>
          <Button variant="secondary" onClick={toggleAlert}>Nevermind</Button>
        </AlertActions>
      </Alert>
    </>
  )
}
```
