## Examples

The Code component uses [react-syntax-highlighter](https://github.com/react-syntax-highlighter/react-syntax-highlighter) with custom [highlight.js](https://highlightjs.org/) themes. Only javascript, css, xml/html, and yml are currently supported.

### Javascript

```jsx example
<Code language="javascript">
  {`// here are some vars
var foo = true;
const number = 1234;
const string = "hello world";
const regex = /^anything$/i;`}
</Code>
```

### HTML

```jsx example
<Code language="html">
  {`<span class="foo">Hello world!</span>`}
</Code>
```

### CSS

```jsx example
<Code language="css">
{`.foo,
#foo,
div[class="foo"] {
  color: green;
}`}
</Code>
```

### No Language Prop
