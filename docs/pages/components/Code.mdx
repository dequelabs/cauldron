## Examples

The Code component uses [react-syntax-highlighter](https://github.com/react-syntax-highlighter/react-syntax-highlighter) with custom [highlight.js](https://highlightjs.org/) themes. Only javascript, css, xml/html, and yml are currently supported.

### Javascript

```jsx example
<Code language="javascript">
  {`// here are some vars
var foo = true;
const number = 1234;
const string = "hello world";
const regex = /^anything$/i;`}
</Code>
```

### HTML

```jsx example
<Code language="html">
  {`<span class="foo">Hello world!</span>`}
</Code>
```

### CSS

```jsx example
<Code language="css">
{`.foo,
#foo,
div[class="foo"] {
  color: green;
}`}
</Code>
```

### No Language Prop

```jsx example
<Code>
```

### Scrollable

By default, Code components will wrap code forcing breaks where it can. If the original formatting of the code needs to be maintained, setting `scrollable` will allow the code block to scroll dynamically adding `tabIndex` to the scrollable area when needed to ensure it is focusable:

```jsx example
<Code scrollable>
 {`<div className="this element has a lot of classnames to ensure it will wrap at longer lengths">
  Hello World
</div>`}
</Code>
```

### Allow Copy

Adds block with label and copy button that allows to copy code to clipboard.

```jsx example
<Code allowCopy label="Click copy button">
 {`<div className="this element has a lot of classnames to ensure it will wrap at longer lengths">
  Hello World
</div>`}
</Code>
```
