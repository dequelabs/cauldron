import { useState } from 'react'
import { ExpandCollapsePanel, PanelTrigger } from '@deque/cauldron-react'

```jsx
import { 
  ExpandCollapsePanel,
  PanelTrigger
} from '@deque/cauldron-react'
```

The `ExpandCollapsePanel` is an unstyled component to provide expand/collapse behavior to a section of content. For a more opinionated component, see [Accordion](./Accordion).

## Examples

### Uncontrolled

By default, the ExpandCollapsePanel is uncontrolled and will store the collapsed state internally with the panel initially collapsed.

```jsx example
<ExpandCollapsePanel>
  <PanelTrigger>There's more info here</PanelTrigger>
  This is content with more info about that thing that you should know.
</ExpandCollapsePanel>
```

### Controlled

If you need to manually control the open/closed state of the panel, you can do this by setting the open prop and handling changes with onToggle.

```jsx example
function ControlledExpandCollapsePanel() {
  const [open, setOpen] = useState(true)
  return (
    <ExpandCollapsePanel open={open} onToggle={() => setOpen(!open)}>
      <PanelTrigger>There's more info here</PanelTrigger>
      This is content with more info about that thing that you should know.
    </ExpandCollapsePanel>
  )
}
```
