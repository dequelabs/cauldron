import { IconButton, Offscreen } from '@deque/cauldron-react'

```js
import { IconButton } from '@deque/cauldron-react'
```

## Examples

### Secondary

Secondary icon buttons are the default variant for the `IconButton` component.

```jsx example
<IconButton icon="pencil" label="Edit" />
<IconButton icon="pencil" label="Edit" disabled />
```

### Primary

Primary icon buttons are used to indicate a primary action.

```jsx example
<IconButton variant="primary" icon="pencil" label="Edit" />
<IconButton variant="primary" icon="pencil" label="Edit" disabled />
```

### Error

Error buttons are used to indicate potentially dangerous actions.

```jsx example
<IconButton variant="error" icon="pencil" label="Edit" />
<IconButton variant="error" icon="pencil" label="Edit" disabled />
```

### Tooltip Customization

The default tooltip can be modified by setting the relevant props configuration from [Tooltip](./Tooltip). This includes all props except for `children` and `target`.

```jsx example
<IconButton 
  icon="trash"
  label="Delete" 
  tooltipProps={{ placement: 'bottom', variant: 'info' }} 
/>
```

### Offscreen Text

If there are multiple icon buttons with the same label, the visible label text can be combined with `Offscreen` to provide additional context to users using assistive technology (AT).

```jsx example
<div>
  User: Jason 
  <IconButton icon="trash" label={<>Delete User <Offscreen>Jason</Offscreen></>} />
</div>
<div>
  User: Harris 
  <IconButton icon="trash" label={<>Delete User <Offscreen>Harris</Offscreen></>} />
</div>
```

### Large

```jsx example
<IconButton variant="error" icon="pencil" label="Edit" large />
<IconButton variant="error" icon="pencil" label="Edit" disabled large />
<IconButton variant="secondary" icon="pencil" label="Edit" large />
<IconButton variant="secondary" icon="pencil" label="Edit" disabled large />
<IconButton variant="primary" icon="pencil" label="Edit" large />
<IconButton variant="primary" icon="pencil" label="Edit" disabled large />
```
