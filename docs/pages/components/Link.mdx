## Examples

### Primary

The default variant for the `Link` component.

```jsx example
<Link>I am a link!</Link>
```

### Button

When you need a semantic link with the visual appearance of a button.

```jsx example
<Link variant="button">I am a link!</Link>
```

### Button (secondary)

When you need a semantic link with the visual appearance of a secondary button. Secondary buttons are used for actions that are not the primary action.

```jsx example
<Link variant="button-secondary">I am a link!</Link>
```

### Additional Link States

A link with visual styling applied to make it look like a button with reduced height.

<div style={{ display: 'flex', flexDirection: 'column' }}>
```jsx example
<Link variant="button" thin>
  I am a thin link!
</Link>

<Link lowercase>I AM A LOWERCASE LINK</Link>

```
</div>
