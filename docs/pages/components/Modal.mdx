## Examples

### Default Modal

By default, modals display with a heading and a dismissable close action:

```jsx example
function DefaultModalExample() {
  const [show, setShow] = useState(false);
  const toggleModal = () => setShow(!show);
  return (
    <>
      <Button variant="secondary" onClick={toggleModal}>
        Toggle Default Modal
      </Button>
      <Modal heading="Default Modal" show={show} onClose={toggleModal}>
        <ModalContent>
          Modal content here, get your modal content here!
        </ModalContent>
        <ModalFooter>
          <Button onClick={toggleModal}>Ok</Button>
        </ModalFooter>
      </Modal>
    </>
  );
}
```

If the action taken from the modal needs to be non-dismissable, use a variant of [Alert](./Alert) instead.

### Info Modal

Alternatively, an info variant is available that is less opinionated in its styling and allows for a more informational presentation of the modal:

```jsx example
function UnstyledModalExample() {
  const [show, setShow] = useState(false);
  const toggleModal = () => setShow(!show);
  return (
    <>
      <Button variant="secondary" onClick={toggleModal}>
        Toggle Info Modal
      </Button>
      <Modal
        heading="Info Modal"
        variant="info"
        show={show}
        onClose={toggleModal}
      >
        <ModalContent>
          Modal content here, get your modal content here!
        </ModalContent>
        <ModalFooter>
          <Button onClick={toggleModal}>Ok</Button>
        </ModalFooter>
      </Modal>
    </>
  );
}
```

### Custom Header Markup

`ModalHeader`, `ModalHeading`, and `ModalCloseButton` are included as custom compound components if you have a use case that requires different header markup.

```jsx example
function CustomHeaderMarkupModalExample() {
  const [show, setShow] = useState(false);
  const toggleModal = () => setShow(!show);
  return (
    <>
      <Button variant="secondary" onClick={toggleModal}>
        Toggle Custom Header Markup Modal
      </Button>
      <Modal variant="info" show={show} onClose={toggleModal}>
        <ModalHeader>
          <div style={{ display: 'flex', flexDirection: 'column' }}>
            <ModalHeading>
              This modal does uses custom header markup!
            </ModalHeading>
            <p style={{ marginLeft: 'var(--dialog-padding)' }}>
              A cool introduction sentence about your modal.
            </p>
          </div>
          <ModalCloseButton />
        </ModalHeader>
        <ModalContent>
          Modal content here, get your modal content here!
        </ModalContent>
        <ModalFooter>
          <Button onClick={toggleModal}>Ok</Button>
        </ModalFooter>
      </Modal>
    </>
  );
}
```
