```js
import { ClickOutsideListener } from '@deque/cauldron-react'
```

## Example

```jsx example
<ClickOutsideListener onClickOutside={() => console.log('You clicked outside!')}>
  <Button>Click Inside</Button>
</ClickOutsideListener>
````
