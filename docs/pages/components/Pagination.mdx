```js
import { Pagination } from '@deque/cauldron-react'
```

## Examples

### Default

```jsx example
<Pagination totalItems={15} />
```

### Thin

```jsx example
<Pagination totalItems={15} thin />
```

### Hiding Start and End Pagination Buttons

Using `hideStartEndPagination` prop, you can hide the "First" and "Last" page buttons.

```jsx example
<Pagination
  totalItems={30}
  hideStartEndPagination
/>
```

### Customizing Pagination Labels

The pagination component sets labels by default, but these labels can be overridden if you need to customize the labels based on the types of items that are being displayed or to provide the labels in a different locale.

```jsx example
<Pagination
  currentPage={1}
  itemsPerPage={1}
  totalItems={25}
  firstPageLabel="First Issue"
  previousPageLabel="Previous Issue"
  nextPageLabel="Next Issue"
  lastPageLabel="Last Issue"
  statusLabel={<span><strong>1</strong> of <strong>25</strong></span>}
/>
```

## Pagination Hook

The pagination component does not maintain any state, but is dependent on state being passed in. This can be done manually using props, or by using the provided `usePagination` hook:

```jsx example
function PaginationWithHook() {
  const { pagination } = usePagination({
    totalItems: 111,
    initialPageSize: 25
  })
  return <Pagination {...pagination} />
}
```

The pagination hook returns an object containing the properties `pagination`, a series of controls that can be spread into the Pagination component, and `pageStatus` which contains details about the state including, `currentPage`, `pageStart`, and `pageEnd`.
